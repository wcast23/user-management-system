# user-management-system
User Management System
A simple user management system implemented in Laravel 10, with CRUD operations (Create, Read, Update, Delete) for managing users in an SQLite database.

Project Overview
This project allows you to:

Create, Read, Update, Delete users.

List all users stored in the SQLite database.

The User entity includes: name, email, phone_number, password, and timestamps.

Prerequisites
Before running the application, ensure you have the following installed:

PHP 8.1+

Composer (for PHP package management)

SQLite (or any other database if you decide to switch)
